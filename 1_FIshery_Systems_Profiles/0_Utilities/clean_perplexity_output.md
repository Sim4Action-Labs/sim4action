# Cleaning Perplexity Output

This script provides instructions for cleaning Perplexity outputs. IMPORTANT: Content must not be modified in any way - only the following specific formatting changes are allowed.

## Cleaning Steps

1. Remove Perplexity Header
   - Delete ONLY these exact lines from the top of the file:
     ```markdown
     <img src="https://r2cdn.perplexity.ai/pplx-full-logo-primary-dark%402x.png" class="logo" width="120"/>

     # 

     ---
     ```
   - DO NOT modify or remove any other content
   - DO NOT add or duplicate any content

2. Amend Title
   - Determine the phase number based on the analysis type:
     * Phase 1: Foundation Elements Analysis
     * Phase 2: Core Variables Analysis
     * Phase 3: System Dynamics
     * Phase 4: Future Pathways
     * Phase 5: Market Analysis and Integration
   - Add ONLY the phase number to the existing title
   - Format as: "[Existing Fishery Name]: Phase X - [Existing Analysis Type]"
   - DO NOT modify any words in the title
   - DO NOT add or remove any other text

3. Format References
   - Move all references to the bottom of the document under a "References" heading
   - Convert each reference from [^X] to [X] format
   - Remove empty lines between references
   - DO NOT modify the reference content or URLs
   - DO NOT add or remove any references

## Quality Checklist

- [ ] Perplexity header removed (ONLY the specified lines)
- [ ] Title amended with phase number (NO other changes to title text)
- [ ] References properly formatted and moved to bottom (NO changes to reference content)
- [ ] NO content modifications made
- [ ] NO content duplicated
- [ ] NO content added or removed except for specified formatting changes
