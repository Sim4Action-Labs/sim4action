# Eastern Tuna and Billfish Fishery (ETBF) Systems Profile - Phase 2: Core Variables Analysis

## Phase Objective
To identify, analyze, and document the core variables that drive the ETBF system across environmental, economic, social, and technical dimensions. This phase establishes measurable indicators and baseline conditions that will form the foundation for understanding system dynamics and future pathways.

## Overview
This document outlines the requirements for Phase 2 of the Eastern Tuna and Billfish Fishery (ETBF) Systems Profile development process. Phase 2 focuses on identifying and analyzing key variables across environmental, economic, social, and technical dimensions. All variables must be expressed in measurable terms that can increase or decrease.

### 4. Technical Variables
Identify and document technical variables that can be measured and tracked over time. Each variable should be expressed in terms that can increase or decrease, with appropriate units of measurement.

## Variable Expression Requirements

### 1. Measurability
- Each variable must be expressed in terms that can be measured
- Include appropriate units of measurement
- Specify measurement frequency or period
- Define measurement methods or data sources

### 2. Directionality
- Variables must be expressed in terms that can increase or decrease
- Avoid ambiguous or qualitative descriptions
- Use quantitative metrics where possible
- For qualitative concepts, develop measurable proxies

### 3. Documentation
- Document the rationale for each variable
- Explain how changes are measured
- Specify data collection requirements
- Note any limitations or uncertainties
