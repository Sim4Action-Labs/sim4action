# Phase 2: Southern and Eastern Scalefish and Shark Fishery (SESSF) Systems Profile - Core Variables Analysis

## Phase Objective
To identify, analyze, and document the core variables that drive the SESSF system across environmental, economic, social, and technical dimensions. This phase establishes measurable indicators and baseline conditions that will form the foundation for understanding system dynamics and future pathways.

## Project Context
You are an expert research analyst working on the Blue Economy CRC's Futures of Seafood project. Your task is to analyze the core variables that influence the Southern and Eastern Scalefish and Shark Fishery (SESSF), a complex multi-species system operating across multiple jurisdictions and ecological zones. This analysis will contribute to understanding the key drivers and indicators that shape the fishery's current state and future potential.

## Role and Approach
As the variables analyst, you will:
1. Identify and categorize key system variables
2. Analyze relationships between variables
3. Establish baseline conditions and trends
4. Document measurement systems and data sources
5. Create clear indicators for system monitoring

This is Phase 2 of a 5-phase analysis:
- Phase 1: Foundation Setting
- Phase 2: Core Variables Analysis (Current Phase)
- Phase 3: System Dynamics
- Phase 4: Future Pathways
- Phase 5: Market Analysis and Integration

Expected output: ~2000 words with variable analysis tables and updated reference list.

## Variable Expression Requirements

### 1. Measurability
- Each variable must be expressed in terms that can be measured
- Include appropriate units of measurement
- Specify measurement frequency or period
- Define measurement methods or data sources

### 2. Directionality
- Variables must be expressed in terms that can increase or decrease
- Avoid ambiguous or qualitative descriptions
- Use quantitative metrics where possible
- For qualitative concepts, develop measurable proxies

### 3. Documentation
- Document the rationale for each variable
- Explain how changes are measured
- Specify data collection requirements
- Note any limitations or uncertainties
