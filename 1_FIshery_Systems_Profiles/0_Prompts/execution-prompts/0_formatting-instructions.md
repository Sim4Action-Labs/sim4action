# Standardized Formatting Instructions for Fishery Systems Profiles

## Document Structure Guidelines

### 1. General Formatting
- Use Markdown formatting throughout all documents
- Maintain consistent heading levels (# for title, ## for major sections, ### for subsections)
- Use bold (**text**) for emphasis of key points
- Use italics (*text*) for technical terms or species names
- Use inline code formatting (`text`) for file names or specific metrics

### 2. Table Formatting
- All tables must use the standard Markdown table format
- Include header row with column names
- Align columns using standard Markdown syntax:
  ```
  | Left-aligned | Center-aligned | Right-aligned |
  |:------------|:-------------:|-------------:|
  | Content     | Content       | Content      |
  ```
- Maintain consistent column widths within each table
- Use consistent units and notation across all tables
- Include table headers and footers where applicable

### 3. Narrative Sections
- Begin each narrative section with a clear topic sentence
- Use paragraphs of 3-5 sentences for readability
- Include transition sentences between major points
- End each narrative section with a summary statement
- Maintain consistent voice and terminology throughout

### 4. Data Presentation
- Use consistent numerical formatting (e.g., two decimal places for percentages)
- Include units of measurement in parentheses
- Present ranges with en dashes (e.g., 10–15)
- Use consistent date formats (YYYY-MM-DD)
- Include confidence levels where applicable (High/Medium/Low)

### 5. Citations and References
- Use inline citations in parentheses (Author, Year)
- Include full references in a dedicated section
- Format references consistently using APA style
- Link to source documents where available
- Cite specific page numbers for direct quotes

### 6. Section-Specific Requirements

#### Executive Summaries
- Length: 300-400 words
- Structure: Context → Key Findings → Implications
- Include quantitative highlights
- End with strategic recommendations

#### Geographic Coverage
- Use consistent geographic naming conventions
- Include coordinates where applicable
- Reference official management areas
- Link to relevant maps or spatial data

#### Stakeholder Analysis
- Use consistent stakeholder categories
- Include influence/interest metrics
- Document relationship dynamics
- Note temporal changes in roles

#### System Variables
- Use consistent variable naming
- Include measurement methods
- Document data sources
- Note confidence levels

#### System Dynamics
- Use consistent relationship notation
- Document feedback loop components
- Include time scales
- Note uncertainty levels

#### Future Pathways
- Use consistent timeframe categories
- Include feasibility assessments
- Document dependencies
- Note implementation barriers

#### Market Integration
- Use consistent market segment definitions
- Include quantitative metrics
- Document competitive factors
- Note market trends

### 8. Quality Control Checklist
- [ ] Consistent formatting throughout document
- [ ] All tables properly formatted and aligned
- [ ] Narratives follow standard structure
- [ ] Data presentation follows guidelines
- [ ] Citations properly formatted
- [ ] Visual elements properly labeled
- [ ] Section-specific requirements met
- [ ] Cross-references verified
- [ ] Units and measurements consistent
- [ ] Terminology consistent throughout

## Implementation Notes
1. These formatting guidelines should be applied to all documents in the fishery systems profiles
2. Any deviations must be explicitly justified
3. Regular review of formatting consistency is required
4. Updates to these guidelines should be version controlled
5. Training materials should reference these standards
