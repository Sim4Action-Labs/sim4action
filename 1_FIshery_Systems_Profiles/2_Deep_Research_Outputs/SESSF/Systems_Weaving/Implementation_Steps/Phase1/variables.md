## Variable Standardization
### Rules Applied
1. Removed directional terms (increasing/decreasing, higher/lower, etc.)
2. Converted to present tense
3. Expressed as measurable quantities
4. Maintained consistency across all variables
5. Preserved original context

### Examples
Original → Standardized
- "Declining stock" → "Stock abundance"
- "Increased fishing effort" → "Fishing effort"
- "Reduced recruitment" → "Recruitment rate"
- "Higher prices" → "Market price"
- "Lower catch rates" → "Catch rate"

### Process
1. Identified variables from source material
2. Removed directional terms
3. Standardized naming convention
4. Added domain categorization
5. Included evidence quality ratings
6. Documented source references
7. Added context descriptions
