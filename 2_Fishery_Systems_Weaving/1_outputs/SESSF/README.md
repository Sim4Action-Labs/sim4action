# Southern and Eastern Scalefish and Shark Fishery (SESSF) Systems Analysis

## Overview
This directory contains the systems analysis workflow for the Southern and Eastern Scalefish and Shark Fishery (SESSF), one of Australia's largest and most diverse fisheries. The analysis follows a structured 9-task approach to create a comprehensive systems map of the fishery.

## Directory Structure
- `tasks/`: Contains the task definitions and templates
- `artifacts/`: Contains the outputs from each task

## Task Sequence
1. Variable Identification
   - Identify 20 key variables representing the SESSF system
   - Document core concepts and measurement units
   - Assess variable frequency and relationships

2. Relationship Extraction
   - Extract direct relationships between variables
   - Document relationship types and strengths
   - Identify temporal and spatial aspects

3. Relationship Matrix
   - Create structured matrix of relationships
   - Identify subsystems and clusters
   - Document matrix completeness

4. Feedback Loop Identification
   - Identify circular relationships
   - Document loop types and strengths
   - Analyze loop implications

5. Indirect Relationship Analysis
   - Analyze chains of relationships
   - Document indirect connections
   - Assess relationship pathways

6. Context-Dependent Relationships
   - Analyze relationship variations
   - Document contextual factors
   - Identify temporal patterns

7. Gap and Uncertainty Analysis
   - Identify knowledge limitations
   - Document uncertainties
   - Assess impact implications

8. Quality Control and Refinement
   - Review system completeness
   - Refine relationships
   - Generate structured JSON output

9. Final System Map Visualization
   - Create force-directed graph
   - Implement interactive features
   - Generate final visualization

## Fishery Context
The SESSF is a multi-species, multi-gear fishery that:
- Operates across southern and eastern Australia
- Targets multiple species of finfish and sharks
- Uses various fishing methods including trawling, longlining, and gillnetting
- Has complex management arrangements
- Faces multiple environmental and economic challenges
